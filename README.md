This project implements a Genetic Algorithm (GA) to solve the Job Shop Scheduling Problem (JSSP)

The objective is to determine the optimal order of jobs that minimizes the overall completion time (makespan) while preventing machine conflicts and deadlocks

A permutation-based chromosome representation is used, where each chromosome represents a possible ordering of jobs. 
The genetic algorithm evolves these schedules through selection, crossover, and mutation operations to identify schedules with lower makespans
